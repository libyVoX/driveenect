# driveenect - проект для drivee в 1-м кейсе Республиканского этапа МПИТ 2026 от команды инж54
Использованные технологии: 
 - React Native от Expo для мобильной разработки
 - Python, uvicorn, FastAPI для серверной части
 - SQLModel для базы данных
#
Структура:
 - /frontend/ - папка с мобильной частью проекта в expo
 - /backend/ - папка серверной части с главной программой, базой данных
 - /backend/routers/ - отдельные модули для обработки серверных запросов
